# Dev Kumar Portfolio Website

A responsive portfolio website built from resume content.

## Files
- `index.html` - main portfolio page
- `styles.css` - styling and layout
- `script.js` - small enhancements

## Run locally
Open `index.html` in a browser.

## Publish with GitHub Pages
1. Create a new repository on GitHub, for example: `dev-kumar-portfolio`
2. Push these files to the repository root
3. In GitHub, open **Settings → Pages**
4. Under **Build and deployment**, select **Deploy from a branch**
5. Choose the `main` branch and the `/root` folder
6. Save changes and wait for the site to publish
